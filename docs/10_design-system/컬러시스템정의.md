# 컬러시스템정의.md

# 1. 문서 목적

본 문서는 미냥이 지갑 서비스의 컬러 시스템을 정의한다.

컬러 시스템은 브랜드 아이덴티티, UI 일관성, 접근성 및 상태 표현을 위한 기준으로 사용한다.

모든 화면 및 컴포넌트는 본 문서의 컬러 시스템을 따른다.

---

# 2. 컬러 설계 목표

## 아기자기한 고양이 컨셉

고양이 캐릭터 기반의 따뜻하고 친근한 느낌을 제공한다.

---

## 가계부 서비스 특성 유지

귀여운 디자인을 유지하되 데이터 가독성을 우선한다.

---

## 장시간 사용 고려

눈의 피로를 최소화할 수 있도록 부드러운 파스텔 계열을 사용한다.

---

# 3. 브랜드 컬러

* 선택 상태

# 10. 카테고리 컬러 규칙

카테고리는 사용자 정의가 가능하므로 고정 컬러를 사용하지 않는다.

기본 제공 카테고리는 다음 팔레트를 순환 사용한다.

```css
#FDA5E3
#FFC1A6
#FFD86B
#8FD19E
#8FB9F7
#BFA8FF
#F7A6C9
#AEE6D8
```

# 14. 다크모드 정책

# 15. 사용 금지 규칙

금지 사항

* 화면마다 임의 색상 사용
* HEX 직접 입력
* 의미 없는 랜덤 컬러 사용
* 상태 색상 임의 변경

모든 UI는 디자인 토큰 및 컬러 시스템을 통해서만 적용한다.
