# 디자인토큰정의.md

# 1. 문서 목적

본 문서는 미냥이 지갑 서비스에서 사용하는 디자인 토큰(Design Token)을 정의한다.

디자인 토큰은 UI를 구성하는 최소 단위 값이며, 디자인 시스템의 기준 데이터로 사용한다.

모든 색상, 여백, 폰트, 그림자, Radius, Animation 값은 본 문서 기준으로 관리한다.

---

# 2. 디자인 토큰 적용 원칙

## 단일 관리 원칙

UI 값은 화면에서 직접 입력하지 않는다.

허용

```css
var(--color-primary)
var(--space-4)
```

금지

```css
#3B82F6
16px
```

---

## 의미 기반 사용

토큰 이름은 역할 기준으로 정의한다.

허용

```css
--color-primary
--color-danger
```

금지

```css
--blue
--red
```

---

## 다크모드 대응

모든 색상은 Semantic Token 기준으로 사용한다.

# 15. 개발 적용 규칙

모든 UI는 디자인 토큰 기반으로 구현한다.

직접 px 값 입력 최소화.
