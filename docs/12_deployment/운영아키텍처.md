# 운영아키텍처.md

# 1. 문서 목적

본 문서는 미냥이 지갑 서비스의 전체 시스템 구조를 정의한다.

본 문서는 개발자가 시스템 전체 구조를 이해하고, 신규 기능 개발 및 유지보수 시 일관된 아키텍처를 유지할 수 있도록 하기 위해 작성한다.

---

# 2. 아키텍처 목표

미냥이 지갑은 다음 목표를 기준으로 설계한다.

## 모바일 우선

```text
Mobile First
```

모든 기능은 모바일 사용성을 우선 고려한다.

---

## Offline First

```text
네트워크 연결 여부와 무관하게
기본 기능 사용 가능
```

---

## PWA 기반

```text
설치형 앱 제공
```

Android / iOS 환경 지원

---

## 확장 가능한 구조

```text
UI 변경
API 변경
기능 추가
```

시 기존 코드 영향 최소화

---

        │

       PostgreSQL
