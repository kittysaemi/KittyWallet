# 3. 환경 변수

## 공통

```env
NODE_ENV=
API_URL=
FRONTEND_URL=
JWT_SECRET=
JWT_REFRESH_SECRET=
DATABASE_URL=
```

---

## 인증/메일

```env
PASSWORD_RESET_URL=
SMTP_HOST=
SMTP_PORT=
SMTP_SECURE=
SMTP_USER=
SMTP_PASSWORD=
SMTP_FROM=
```

- `PASSWORD_RESET_URL`은 비밀번호 재설정 이메일에 포함할 프론트엔드 URL이다.
- SMTP 비밀값은 Git에 저장하지 않고 환경 변수, Secret Store, CI/CD Secret으로 관리한다.

---


# 4. 환경별 정책

## Local

* 로컬 DB 허용
* 디버그 로그 허용

---

## Dev

* 테스트 데이터 허용
* 개발 로그 허용

---

## Staging

* 운영과 동일 환경
* 테스트 계정 사용

---

## Production

* 운영 데이터 사용
* Debug 로그 금지
