## OP-CHECK-003

### 항목

Clean DB Migration 순서 검증

### 수행 방법

비어 있는 PostgreSQL 데이터베이스를 기준으로 전체 Prisma migration을 처음부터 적용한다.

```bash
npm run db:migrate
```

### 기대 결과

```text
All migrations have been successfully applied.
```

### 실패 시 조치

실패한 migration보다 뒤에 있는 참조 테이블 생성 migration이 있는지 확인하고, migration 적용 순서를 수정한다.
