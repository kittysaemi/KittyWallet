운영 환경 배포 절차를 표준화한다.

---

# 2. 배포 흐름

```text
개발 완료
↓
PR 생성
↓
코드 리뷰
↓
QA 수행
↓
E2E 수행
↓
릴리즈 생성
↓
Docker Build
↓
Docker Push
↓
운영 배포
↓
Health Check
↓
운영 검수
```

# 3. 배포 준비

## 코드 확인

* main 최신 반영
* 테스트 통과
* 코드 리뷰 완료

## 문서 확인

* 변경 이력 작성
* 릴리즈 노트 작성

---

# 4. Docker Build

```bash
docker build -t kittywallet-api:1.0.0 .
```

---

# 5. Docker Push

```bash
docker push registry/kittywallet-api:1.0.0
```

---

# 6. 서버 배포

```bash
docker compose pull
docker compose up -d
```

---

# 7. DB Migration

```bash
npx prisma migrate deploy
```

---

# 8. Health Check

## API

```http
GET /health
```

## 확인 항목

* 로그인
* 거래 등록
* 거래 조회
* 통계 조회
