# 오류 코드 정의

## 코드 형식

사용자 화면과 API 응답에 노출되는 오류 코드는 특수문자 없는 5자리 영숫자(`^[A-Z0-9]{5}$`)를 사용한다. 첫 글자는 오류 그룹, 뒤 4자리는 그룹 내 일련번호다. 그룹의 9,999개를 모두 사용하면 새 코드부터 6자리 이상으로 확장할 수 있으며, 이미 발급한 5자리 코드는 변경하지 않는다.

백엔드 내부 식별자는 구현·로그 검색을 위해 유지하지만 외부 응답에는 공개 코드만 사용한다. 신규 오류는 `packages/shared-types/src/errorCodes.ts`와 이 문서에 함께 등록한다.

개별 API 문서의 기존 오류 표에 있는 밑줄 형식 값은 내부 식별자이며, 사용자 화면과 API 응답의 공개 코드는 이 문서의 `공개 코드`를 기준으로 한다.

## 화면 오류 표시 정책

등록 화면을 포함한 사용자 화면은 서버 상세 메시지를 직접 표시하지 않고 다음 공통 형식을 사용한다.

```text
오류입니다 #{publicCode} 관리자에 문의하세요.
```

사용자가 즉시 수정할 수 있는 입력 필드 검증 메시지는 이 정책의 대상이 아니다.
