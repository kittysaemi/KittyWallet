# 사이트맵

## 1. 사이트맵 개요

미냥이 지갑은 브라우저 기반 PWA 모바일 웹앱 서비스이며, 사용자가 모바일 환경에서 빠르게 거래를 기록하고 자산 흐름을 관리할 수 있도록 구성한다.

전체 구조는 다음 기준으로 설계한다.

* 모바일 우선 구조
* 브라우저 기반 실행 지원
* 홈 화면 설치 지원
* 최소 Depth 기반 빠른 접근
* 거래 입력 중심 UX
* 계좌/카드/카테고리 분리 관리
* 통계 및 검색 기능 제공
* 인증 기반 접근 제어
* 오프라인 일부 기능 지원

# 4. 공통 화면 이동 규칙

## 인증 필요 화면

로그인 이후 화면은 모두 인증 상태를 필요로 한다.

비인증 상태 발생 시:

* 로그인 화면 이동
* 토큰 초기화
* 사용자 상태 초기화

---

## 자동 로그인 처리

앱 실행 시 JWT 상태를 확인한다.

* 유효한 경우 메인 대시보드 이동
* 만료된 경우 로그인 화면 이동

---

## 아이콘 선택 화면

아이콘 선택 화면은 다음 화면에서 공통 사용한다.

* 계좌 등록
* 카드 등록
* 카테고리 등록

---

## 거래 등록 접근

거래 등록 화면은 다음 위치에서 접근 가능하다.

* 메인 대시보드
* 거래 검색 결과
* 최근 거래 목록

---

## PWA 설치 접근

PWA 설치 기능은 브라우저 환경에서 제공한다.

설치 가능 상태일 경우:

* 설치 배너 노출
* 홈 화면 설치 유도
