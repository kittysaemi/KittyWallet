# 사용자 플로우

## 1. 사용자 플로우 개요

미냥이 지갑은 브라우저 기반 PWA 모바일 웹앱 서비스이며, 사용자가 모바일 환경에서 빠르게 거래를 기록하고 자산 흐름을 관리할 수 있도록 설계한다.

사용자 플로우는 다음 기준을 기반으로 구성한다.

* 브라우저 기반 실행 지원
* 모바일 우선 UX
* 최소 이동 기반 구조
* 빠른 거래 입력 중심
* 상태 기반 처리
* 인증 기반 접근 제어
* 오프라인 환경 대응 고려
* 홈 화면 설치 지원
* 유지보수 가능한 흐름 구조 유지
