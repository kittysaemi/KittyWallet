# 기술 스택

## 1. 문서 목적

본 문서는 미냥이 지갑 프로젝트에서 사용하는 기술 스택과 선택 기준을 정의한다.

기술 스택은 단순 라이브러리 사용 목록이 아니라 다음 목표를 기준으로 선정한다.

* 유지보수 가능한 구조
* 확장 가능한 구조
* 모바일 중심 사용자 경험
* PWA 기반 앱 구조
* API 변경 대응 구조
* 상태 기반 UI 처리
* AI 협업 가능한 개발 환경

---
