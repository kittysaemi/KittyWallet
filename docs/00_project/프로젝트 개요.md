# 프로젝트 개요

## 1. 프로젝트명

미냥이 지갑 (KITTYWALLET)

---

## 2. 프로젝트 설명

미냥이 지갑은 계좌, 카드, 거래 내역을 통합 관리할 수 있는 PWA 기반 모바일 웹앱 서비스이다.

사용자는 모바일 브라우저 환경에서 서비스를 사용할 수 있으며, 홈 화면 설치를 통해 앱처럼 사용할 수 있다.

서비스는 거래 기록, 거래 검색, 소비 통계, 계좌 관리, 카드 관리, 카테고리 관리 기능을 중심으로 구성한다.

현재는 HTML/CSS/JavaScript 기반 localStorage 프로토타입 형태로 구성되어 있으며, 향후 API 기반 구조 및 Capacitor 기반 하이브리드 앱 구조로 확장하는 것을 목표로 한다.

---

## 3. 프로젝트 목적

- 모바일 중심 가계부 서비스 제공
- 빠른 거래 입력 환경 제공
- 계좌/카드/거래 내역 통합 관리
- 소비 흐름 및 통계 정보 제공
- 오프라인 사용 가능한 PWA 구조 제공
- 유지보수 가능한 구조 기반 개발 환경 구성
- API 및 기능 변경에 유연하게 대응 가능한 구조 설계
- 장기 운영 가능한 모바일 웹앱 구조 구성

# 5. 대상 사용자

* 개인 자산 및 소비 흐름을 관리하려는 사용자
* 빠르게 거래 내역을 기록하려는 사용자
* 모바일 중심으로 가계부를 사용하는 사용자
* 계좌/카드 사용 내역을 통합 관리하려는 사용자

## 6. 핵심 기능

### 사용자 기능
- 회원가입
- 로그인 및 인증
- 비밀번호 재설정
- 자동 로그인
- 사용자 설정 관리

### 거래 관리
- 수입/지출 거래 등록
- 거래 수정 및 삭제
- 거래 검색
- 기간별 조회

### 계좌 관리
- 계좌 등록 및 관리
- 잔액 관리

### 카드 관리
- 카드 등록 및 관리
- 카드별 사용 내역 확인

### 카테고리 관리
- 카테고리 등록 및 수정
- 아이콘 선택 기능 제공

### 통계 기능
- 소비 통계 확인
- 기간별 소비 흐름 분석

### PWA 기능
- 모바일 브라우저 실행 지원
- Web App Manifest 지원
- Service Worker 기반 캐시 처리
- 홈 화면 설치 지원
- 오프라인 일부 기능 지원

# 8. 개발 방향

본 프로젝트는 단순 화면 중심 개발이 아니라 구조 중심 개발을 목표로 한다.

다음 원칙을 기준으로 개발한다.

* 기능과 API 의존성 분리
* 프론트엔드/백엔드 계층 분리
* Repository 기반 데이터 접근
* 정책 기반 기능 구현
* 상태 기반 UI 처리
* 예외 처리 우선 설계
* 유지보수 및 확장성 우선 고려

또한 GitHub 기반 문서 관리 및 AI(Codex/GPT) 협업 환경을 기준으로 개발을 진행한다.
