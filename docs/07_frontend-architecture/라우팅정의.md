# 라우팅 정의

## 1. 문서 목적

본 문서는 미냥이 지갑 프론트엔드 라우팅 구조를 정의한다.

---

# 2. 라우팅 목표

- 화면 구조 명확화
- 인증 분리
- Lazy Loading 지원
- 모바일 탐색 단순화

---

# 3. 기본 구조

```text
/login
/dashboard
/transactions
/accounts
/cards
/categories
/statistics
/settings
```

---

# 4. 인증 라우트

| 경로 | 설명 |
|---|---|
| /login | 로그인 |
| /signup | 회원가입 |
| /reset-password | 비밀번호 재설정 |

---

# 5. 거래 라우트

| 경로 | 설명 |
|---|---|
| /transactions | 거래 목록 |
| /transactions/new | 거래 등록 |
| /transactions/:id | 거래 상세 |
| /transactions/:id/edit | 거래 수정 |

---

# 6. 계좌 라우트

| 경로 | 설명 |
|---|---|
| /accounts | 계좌 목록 |
| /accounts/new | 계좌 등록 |
| /accounts/:id/edit | 계좌 수정 |

---

# 7. 카드 라우트

| 경로 | 설명 |
|---|---|
| /cards | 카드 목록 |
| /cards/new | 카드 등록 |
| /cards/:id/edit | 카드 수정 |

---

# 8. 카테고리 라우트

| 경로 | 설명 |
|---|---|
| /categories | 카테고리 목록 |
| /categories/new | 카테고리 등록 |
| /categories/:id/edit | 카테고리 수정 |

---

# 9. 아이콘 라우트

| 경로 | 설명 |
|---|---|
| /icons | 아이콘 목록 |
| /icons/new | 아이콘 등록 |
| /icons/:id/edit | 아이콘 수정 |
| /icons/select | 아이콘 선택 (계좌/카드/카테고리 등록 시 공통 사용) |

---

# 10. 통계 라우트

| 경로 | 설명 |
|---|---|
| /statistics | 통계 메인 (월별/카테고리별/기간별) |

---

# 11. 설정 라우트

| 경로 | 설명 |
|---|---|
| /settings | 사용자 설정 |

---

# 12. 라우팅 처리 규칙

- 인증 필요 페이지 보호 처리
- Lazy Import 사용
- 화면 단위 코드 분리
- 모바일 뒤로가기 대응

---

# 13. 예외 처리

| 상황 | 처리 |
|---|---|
| 인증 만료 | 로그인 이동 |
| 존재하지 않는 페이지 | 404 |
| 권한 없음 | 접근 차단 |
