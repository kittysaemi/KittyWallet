# 계좌정책

## 1. 문서 목적

본 문서는 계좌 등록 및 잔액 관리 기준을 정의한다.

---

# 2. 계좌 기본 정책

| 항목     | 정책              |
| ------ | --------------- |
| 계좌 등록  | 사용자 직접 등록       |
| 계좌명 중복 | 동일 사용자 기준 중복 불가 |
| 초기 잔액  | 입력 가능           |
| 현재 잔액  | 거래 기준 자동 계산     |
| 사용 여부  | ON/OFF 지원       |

---

# 3. 잔액 정책

| 항목       | 정책                                         |
| -------- | ------------------------------------------ |
| 수입 거래    | 잔액 증가                                     |
| 지출 거래    | 잔액 감소                                     |
| 거래 수정    | 차액 기준 재계산                                 |
| 거래 삭제    | 잔액 복구 처리                                  |
| 잔액 음수 허용 | 허용하지 않음                                   |
| 잔액 부족 처리 | 지출 등록/수정 후 잔액이 0 미만이 되는 경우 사용자에게 알리고 등록 차단 |

---

# 4. 아이콘 정책

| 항목     | 정책           |
| ------ | ------------ |
| 아이콘 지정 | 가능           |
| 아이콘 선택 | 공통 아이콘 화면 사용 |

---

# 5. 비활성 정책

| 항목       | 정책          |
| -------- | ----------- |
| use_yn=false | 선택 목록 제외 가능 |
| 거래 연결 계좌 | 물리 삭제 금지    |
| 삭제 대신    | 비활성 처리 사용   |
