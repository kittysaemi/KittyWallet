# 카테고리정책

## 1. 문서 목적

본 문서는 카테고리 관리 및 거래 연동 기준을 정의한다.

---

# 2. 카테고리 기본 정책

| 항목      | 정책        |
| ------- | --------- |
| 카테고리 등록 | 사용자 직접 등록 |
| 카테고리 구조 | 1Depth 구조 |
| 카테고리 수정 | 명칭 수정 가능  |
| 아이콘 지정  | 가능        |

---

# 3. 표시 정책

| 항목     | 정책            |
| ------ | ------------- |
| 표시 여부  | show=true만 표시 |
| 기본 정렬  | 등록 순서 기준      |
| 표시 OFF | 선택 목록 제외      |

---

# 4. 거래 연동 정책

| 항목         | 정책         |
| ---------- | ---------- |
| 거래 등록      | 카테고리 선택 필수 |
| 삭제된 카테고리   | 기존 거래 유지   |
| 거래 연결 카테고리 | 물리 삭제 금지   |

---

# 5. 삭제 정책

| 항목          | 정책            |
| ----------- | ------------- |
| 삭제 처리       | show=false 사용 |
| Soft Delete | 우선 적용         |
