# 카드정책

## 1. 문서 목적

본 문서는 카드 등록 및 카드 거래 처리 기준을 정의한다.

---

# 2. 카드 기본 정책

| 항목       | 정책           |
| -------- | ------------ |
| 카드 등록    | 사용자 직접 등록    |
| 카드 사용 여부 | ON/OFF 지원    |
| 카드 아이콘   | 지정 가능        |
| 목록 표시    | 사용중 카드 우선 표시 |

---

# 3. 거래 연동 정책

| 항목    | 정책                     |
| ----- | ---------------------- |
| 거래 저장 | `transaction_type=EXPENSE`, `wallet_type=CARD` 기준 저장 |
| 수입 거래 | 카드는 수입이 없으므로 수입 선택 시 카드 선택 불가 |
| 통계 반영 | 카드 거래는 계좌 거래와 독립 관리하며 지출 통계에 포함 가능 |
| 카드 거래 | TRANSACTION 기준 관리      |
| 계좌 잔액 | 카드 거래 저장/수정/삭제 시 계좌 잔액 변경 없음 |

---

# 4. 삭제 정책

| 항목       | 정책          |
| -------- | ----------- |
| 거래 연결 카드 | 물리 삭제 금지    |
| 삭제 처리    | use_yn=false 처리 |
| 비활성 카드   | 선택 목록 제외    |
