# 캐시정책

## 1. 문서 목적

본 문서는 미냥이 지갑 PWA의 캐시 저장 기준, 캐시 전략, 만료 정책, 갱신 정책 및 무효화 기준을 정의한다.

캐시는 단순 성능 향상 목적뿐 아니라 다음 기능을 지원하기 위해 사용한다.

* 빠른 앱 실행
* 오프라인 조회 지원
* 네트워크 장애 대응
* 서버 부하 감소
* 사용자 경험 향상

본 문서는 다음 문서와 연계된다.

* PWA개요
* ServiceWorker정의
* 오프라인동작정의
* 동기화구조
* 업데이트정책
* PWA상태정의

---

# 2. 캐시 정책 개요

미냥이 지갑은 PWA 기반 서비스로 다음 원칙을 따른다.

### 원칙 1

앱 실행에 필요한 리소스는 항상 캐시한다.

### 원칙 2

조회 데이터는 최근 데이터 기준으로 캐시한다.

### 원칙 3

사용자 입력 데이터는 캐시가 아닌 IndexedDB를 사용한다.

### 원칙 4

민감 정보는 캐시에 저장하지 않는다.

### 원칙 5

새 버전 배포 시 캐시는 자동 정리한다.

---

# 3. 캐시 저장소 정의

미냥이 지갑은 다음 저장소를 사용한다.

| 저장소           | 용도             |
| ------------- | -------------- |
| Cache Storage | 앱 리소스 및 조회 데이터 |
| IndexedDB     | 오프라인 거래 저장     |
| localStorage  | PWA 실행 플래그 (설치 유도 상태 등 비민감 값만) |

> **사용자 설정은 localStorage에 저장하지 않는다.**
> 사용자 설정은 서버 `USER_SETTING` 테이블 기준으로 관리하며 `GET /api/v1/settings`, `PUT /api/v1/settings` API를 통해 조회 및 저장한다.

---

## Cache Storage

대상

```text
HTML
CSS
JavaScript
Font
Icon
API Response
```

---

## IndexedDB

대상

```text
거래 임시 저장 (offline_transactions)
Sync Queue (sync_queue)
동기화 이력 (sync_history)
클라이언트 식별 정보 (sync_client)
PWA 앱 설정 플래그 (app_settings)
```

> **API 조회 데이터(최근 거래, 대시보드, 통계, 계좌/카드/카테고리 목록)는 IndexedDB에 저장하지 않는다.**
> API response 캐시는 Cache Storage(Workbox Runtime Cache)에서 관리한다.
> TanStack Query의 `persistQueryClient`는 MVP에서 사용하지 않는다.

---

## localStorage

대상

```text
install_prompt_dismissed  (설치 안내 닫힘 여부)
standalone_launch_hint    (홈 화면 설치 유도 표시 여부)
```

> localStorage는 위와 같은 **비민감 PWA 실행 플래그**만 저장한다.
> 사용자 설정(테마, 언어, 알림 등)은 서버 `USER_SETTING`에 저장하고 `GET/PUT /api/v1/settings`로 처리한다.
> 토큰, 사용자 정보, 거래 데이터는 저장하지 않는다.

# 5. 캐시 전략 정의

## Cache First

캐시 우선

```text
Cache 조회
 ↓
존재
 ↓
즉시 반환

없음
 ↓
Network 요청
 ↓
Cache 저장
```

---

적용 대상

```text
App Shell
Icon
Font
```

---

선정 이유

```text

실행 속도 중요
```

---

```text

 ↓

 ↓

 ↓
```

| 기능              | 저장소           |

# 16. 최종 목표

미냥이 지갑의 캐시는 단순 성능 향상이 아니라 오프라인 사용성과 데이터 안정성을 지원하기 위한 핵심 기능이다.

최종적으로 다음을 만족해야 한다.

* 빠른 앱 실행
* 최근 데이터 조회
* 오프라인 조회 지원
* 자동 캐시 갱신
* 데이터 무결성 유지
* 버전 업데이트 대응
* 사용자 경험 향상
