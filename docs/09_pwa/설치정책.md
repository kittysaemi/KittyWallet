# 설치정책

## 1. 문서 목적

본 문서는 미냥이 지갑의 PWA 설치 기능에 대한 처리 기준과 사용자 경험 정책을 정의한다.

설치정책은 다음 문서와 연계된다.

* PWA개요
* Manifest정의
* ServiceWorker정의
* PWA상태정의
* 화면정의
* 업데이트정책

본 정책은 브라우저 기반 서비스가 모바일 앱과 유사한 설치 경험을 제공하기 위한 기준 문서로 사용한다.

---

# 2. 설치 정책 개요

미냥이 지갑은 브라우저 기반 서비스이지만 사용자가 홈 화면에 설치하여 모바일 앱처럼 사용할 수 있어야 한다.

이를 위해 PWA 설치 기능을 제공한다.

설치 기능의 목적은 다음과 같다.

* 앱 수준 사용자 경험 제공
* 빠른 실행 지원
* 브라우저 주소창 제거
* 전체 화면 실행
* 재방문 사용성 향상
* 홈 화면 접근성 향상

# 4. 설치 상태 정의

설치 기능은 다음 상태를 기준으로 동작한다.

| 상태          | 설명             |
| ----------- | -------------- |
| installable | 설치 가능          |
| installed   | 설치 완료          |
| dismissed   | 사용자가 설치 안내를 닫음 |
| unsupported | 설치 미지원 환경      |

---

## installable

설치 가능 상태이다.

조건

* Manifest 로드 성공
* Service Worker 등록 완료
* 브라우저 설치 조건 충족

처리

* 설치 배너 표시 가능
* 설치 버튼 활성화

---

## installed

설치 완료 상태이다.

조건

* 홈 화면 설치 완료
* standalone 모드 실행

처리

* 설치 배너 숨김
* 설치 안내 비노출

---

## dismissed

사용자가 설치 안내를 닫은 상태이다.

처리

* 즉시 재노출 금지
* 일정 기간 이후 재노출

---

## unsupported

브라우저가 설치를 지원하지 않는 상태이다.

처리

* 설치 버튼 숨김
* 설치 안내 미표시

---

# 5. 설치 가능 조건

설치 가능 여부는 브라우저 기준을 따른다.

필수 조건

```text
Manifest 존재
AND
Service Worker 등록
AND
HTTPS 사용
AND
설치 미완료
```

모든 조건 충족 시 installable 상태로 진입한다.

---

# 6. 설치 배너 정책

## 노출 시점

다음 조건 충족 시 설치 배너를 표시한다.

```text
installable 상태
AND
로그인 완료
AND
설치 미완료
```

---

## 노출 위치

기본 위치

* 메인 대시보드 상단

보조 위치

* 사용자 설정 화면

---

## 배너 구성

표시 항목

* 앱 이름
* 앱 아이콘
* 설치 안내 문구
* 설치 버튼
* 닫기 버튼

예시

```text
미냥이 지갑을 홈 화면에 설치해보세요.

[설치하기]
[나중에]
```

---

# 7. 설치 요청 정책

사용자가 설치 버튼을 클릭한 경우

```text

 ↓
사용자 승인
 ↓
설치 완료
 ↓
installed 상태 전환
```

---

## 설치 성공

처리

* installed 상태 변경
* 설치 배너 제거
* 설정 정보 저장

---

## 설치 취소

처리

* installable 유지
* 설치 배너 재노출 가능

---

# 8. 설치 완료 정책

설치 완료 후 다음 기준을 적용한다.

## 배너 처리

```text
설치 완료

배너 제거
 ↓
재노출 금지
```

---

## 실행 방식

```text

```

---

## 사용자 경험

적용 항목

* 브라우저 주소창 제거
* 전체 화면 실행
* 앱 아이콘 사용
* Splash 화면 사용

---

# 9. iOS 설치 정책

Safari iOS는 브라우저 설치 API를 제공하지 않는다.

따라서 별도 안내를 제공한다.

## 설치 절차

```text
Safari 실행
 ↓
공유 버튼 선택
 ↓
홈 화면에 추가
 ↓
설치 완료
```

---

## 안내 정책

설치 가능 시

사용자에게 단계별 설치 안내를 제공한다.

예시

```text
공유 버튼을 눌러
홈 화면에 추가를 선택해주세요.
```

---

# 10. 설치 안내 재노출 정책

사용자가 설치 배너를 닫은 경우

dismissed 상태로 처리한다.

---

## 재노출 기준

| 조건    | 처리     |
| ----- | ------ |
| 1일 미만 | 재노출 안함 |
| 1일 이상 | 재노출 가능 |
| 설치 완료 | 재노출 안함 |

---

## 저장 위치

```text
localStorage
```

저장 항목

```json
{
  "installDismissedAt": "2026-01-01T12:00:00"
}
```
